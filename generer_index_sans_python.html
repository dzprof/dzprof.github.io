Générateur rapide de journaux_index.json

Cette page crée le fichier journaux_index.json directement dans le navigateur, sans Python. Elle lit les PDF du dossier JOURNAUX, extrait les ISSN/titres, puis télécharge le fichier JSON.

1) Cliquez sur le bouton ci-dessous et sélectionnez le dossier JOURNAUX complet.

Le dossier doit contenir par exemple : JOURNAUX/2018/A/, JOURNAUX/2018/B/, JOURNAUX/2018/A+/.

Sélectionnez d'abord le dossier JOURNAUX.
Aucun fichier sélectionné.

2) Après téléchargement, placez journaux_index.json dans la racine de votre site GitHub Pages, à côté de index.html. Ensuite, rechargez le site avec Ctrl + F5.
