2.1 Enseignement : cours / TD / TP

Règle appliquée : saisissez uniquement le nombre de semestres pour chaque activité. Le calcul respecte la priorité Cours > TD > TP : si un cours est déclaré pour un semestre, le TD/TP du même semestre n’est pas ajouté ; si TD et TP existent sans cours, seul le TD est compté. Le total Cours + TD + TP est plafonné à 60 pts.

3) Articles scientifiques A+ / A / B

Vérification automatique par ISSN

Pour chaque article : sélectionnez l’année de publication, saisissez l’ISSN de la revue ou, facultativement, le titre de la revue / éditeur pour les listes prédatrices sans ISSN, puis indiquez votre rang d’auteur. Le calcul utilise : catégorie A+ = 160 pts, catégorie A = 80 pts, catégorie B = 40 pts, 1er auteur = 100 %, 2e auteur = 50 %, 3e auteur et plus = 25 %. Si la revue est dans un fichier prédatrice/prédateur, le score est 0 et le résultat indique “Revue prédatrice”.
